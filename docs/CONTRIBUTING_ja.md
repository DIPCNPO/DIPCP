# DIPCP に貢献する

DIPCP プロジェクトへの貢献をご検討いただき、ありがとうございます！私たちは、**「多くの知恵を集めれば、後世に残る優れた作品を創作できる」** と信じています。恒久的で協力的なエコシステムを提供することで、世界中の作家やクリエイターが、個人の力だけでは達成できない目標を実現できるようにします。このエコシステムでは、個人の想像力が集団の知恵によって増幅され、優れた知的財産権を創造することが可能になります。

## 私たちの理念

* **市場自治（Market Autonomy）：** コンテンツの価値は、いかなる委員会や個人の主観的評価ではなく、コミュニティによる**引用回数**と**純「いいね」数**によって決定される。
* **フォークの権利化（Forking as Rights）：** 貢献者は、自身の制作物が**採用パス**内で**修正、終了、または削除**される可能性があることを受け入れ、それを阻止する権利を有しない。
* **市場の評価に基づく利益獲得：** 読者は各記事に対して投票し、**「いいね」**、**「よくないね」**、または**「パス」**することができます。作者は互いの記事を引用し合うことができます。商業運用時には、記事の**有効文字数**、**引用回数**、**純「いいね」数**に基づいて、各作者の分配比率が決定されます。

### 基本原則

**【絶対的前提】** すべての貢献者は、**貢献者ライセンス同意書（CLA）** に署名し、以下のことを**取消不能に**宣言しなければならない：

1.  **永久的許諾：** すべてのコンテンツは、DIPCNPO（分散型IPコラボレーション非営利組織）に対して永久的かつ取消不能に許諾され、同組織が商業運営を一括して担当する。
2.  **運命の放棄：** 貢献者は、自身の制作物の役割、設定、または筋書きが、**採用パス**内で**後の作者によって修正または終了される可能性がある**ことを受け入れ、そのキャラクターの **「生存権」と「最終的な運命」** に対する管理権を放棄する。

## 良い創作とは何か？

各記事は、共同創作に価値を提供すべきです。自問してください：**「これはこの物語をさらに面白くしているか？」**

**✅ 方向性に合致する創作：**

* 不足しているコンテンツの補完
* 誤字の修正
* より引き込まれるストーリー
* より豊富なプロット
* 他の言語への翻訳

**❌ 方向性に合致しない創作：**

* 他者の記事を誹謗中傷するもの
* 作品の世界観に合わない内容

## 必ず注意すること

1.  自分の記事で、他人の記事の段落を大量に引用する場合、引用した段落は `[]` 角括弧で囲み、その後ろを `()` 丸括弧で囲んで原文のパスを記載する必要があります。
2.  パスの形式は `(作者/作品/ディレクトリ/ファイル)` です。
3.  引用を明示しない場合、内容は盗用とみなされ、最終的な商業化審査時、記事全体が採用されず、報酬の対象外となります。
4.  内容が他の記事から盗用された場合、最終的な商業化審査時、記事全体が報酬計算の対象から外れるだけでなく、法的訴訟に直面する可能性もあります。
5.  あなたの記事は、他の人に引用されて初めて読者に見てもらえ、商業的価値を生み出すことができます。

**覚えておいてください**：すべての偉大な文豪は、かつては初心者でした。私たちと一緒に、後世に語り継がれる物語を創作しましょう。
