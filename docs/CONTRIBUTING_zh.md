# 为 DIPCP 做贡献

感谢你考虑为 DIPCP 项目做出贡献！我们相信 **“群策群力才能创作出流芳百世的优秀作品”**， 通过提供一个永久的、协作的生态系统，使全球范围内的作家和创作者能够实现仅凭个人力量无法完成的目标。在这个生态系统中，个体的想象力被集体的智慧放大，从而能够创造出优秀的知识产权。

## 我们的理念

* **市场自治（Market Autonomy）：** 内容的价值由社区的**引用次数**和**净赞数**决定，而非任何委员会或个人主观评定。
* **分叉即权利（Forking as Rights）：** 贡献者必须接受其创作在被**采纳路径**中可能被**修改、终结或移除**，且无权阻止。
* **根据市场认可度获利：** 读者要为每篇文章投票，可以**点赞**、**点踩**或**路过**，作者可以相互引用彼此的文章。商业运营时，将根据文章的**有效字数**、**引用次数**和**净赞数**决定每位作者的分成比例。

### 核心原则

**【绝对前提】** 所有贡献者必须签署**贡献者许可协议（CLA）**，并**不可撤销地**声明：

1.  **永久许可：** 所有内容被永久、不可撤销地授权给DIPCNPO（去中心化IP协作非盈利组织），由其统一负责商业运营。
2.  **命运放弃：** 贡献者必须接受其创作的角色、设定或故事情节，在被**采纳路径**中**可以被后续作者修改或终结**，并放弃对该角色的 **“生存权”和“最终命运”**的控制权。

## 什么是好的创作？

每篇文章都应该为协同创作提供价值。问问自己：**“这是否让这个故事更加有趣？”**

**✅ 符合方向的创作：**

* 弥补缺失的内容
* 修改错别字
* 更引人入胜的故事
* 更丰富的情节
* 翻译为其他语言

**❌ 不符合方向的创作：**

* 诋毁、辱骂别人的文章
* 不符合作品世界观的内容

## 必须注意

1. 如果要在你的文章中大段引用别人文章中的段落，引用的段落必须用[]方括号包裹，后面用()圆括号包括原文路径
2. 路径格式为(作者/作品/目录/文件)
3. 如果不指明引用，内容将被视为抄袭，最终商业化审核时，整篇文章都不会被采纳，无法计酬
4. 如果内容抄袭自其它文章，最终商业化审核时，整篇文章都无法纳入计酬范围，还有可能面临司法起诉
5. 你的文章只有被别人引用才能被读者看到，才会产生商业价值

**请记住**：每位大文豪曾经都是新手。让我们一起来创作能够流芳百世的故事吧。
