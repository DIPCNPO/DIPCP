DIPCP is a fully transparent, decentralized, serverless collaboration platform. All data is publicly stored in GitHub repositories. Please read this Usage Agreement carefully.

## Transparency Principle:

DIPCP adheres to a fully transparent design principle. All participants, participation processes, and benefits received are explicitly and publicly disclosed, with full traceability throughout.

## 1. Platform Overview

DIPCP (Decentralized IP Collaboration Platform) is a GitHub-based collaboration platform specifically designed for decentralized world-building, content creation, and collaborative projects. This platform follows a fully transparent design philosophy. Apart from the CLA (Contribution License Agreement) signed by users, all data is stored publicly.

**Important Note:** DIPCP itself does not provide any services; all services are provided by GitHub. We only provide a client-side application to help users better utilize GitHub's collaboration features.

## 2. Applicable Scenarios

### 2.1 Suitable Project Types

- **World-Building Projects**: Game settings, novel worldviews, and other creative content.

### 2.2 Unsuitable Project Types

#### Important Reminder:

The following types of projects are **not suitable** for using DIPCP:

- **Commercial Confidential Projects**: Internal projects involving trade secrets or commercial confidentiality.
- **Personal Private Content**: Sensitive information involving personal privacy.
- **Confidential Projects**: Projects requiring non-disclosure agreements (NDAs).
- **Internal Management Systems**: Enterprise internal management systems.
- **Healthcare Projects**: Projects involving personal health information.
- **Financial Service Projects**: Projects involving financial privacy.
