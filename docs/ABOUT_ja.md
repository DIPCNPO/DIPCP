# **プロジェクト紹介：DIPCP——分散型知的財産協作プラットフォーム**
# **（DIPCP: Decentralized Intellectual Property Collaboration Platform）**

## **一、プロジェクトの位置付けとビジョン**

### **1. 組織主体：DIPCNPO (分散型知的財産協作非営利法人)**

* **性質：** シンガポールに設立された非営利法人組織。
* **使命：** 分散した知的財産（IP）を集中管理し、世界中のクリエイターに、安定した、法的に健全で、かつ永久的に無料の協作環境を提供すること。
* **ビジョン：** 世界中のクリエイターに力を与え、協働の力によって後世に残る偉大な知的財産作品を共同で創作すること。

### **2. 中核プラットフォーム：DIPCP**

DIPCPは、**GitHub Pages** を基盤とする分散型協作プラットフォームであり、従来の協作モデルにおける**高コスト、IP帰属権の曖昧さ、中央集権的な検閲**という課題を根本的に解決します。プラットフォームは **「データが審判である」** というインセンティブメカニズムを採用し、クリエイターがフォーク、引用、市場評価を通じてIP価値を育成することを奨励します。

## **二、革新的な技術と運用アーキテクチャ**

DIPCF財団は、ミニマリズムと分散技術を活用し、前例のないゼロコスト運営モデルを構築し、極限の透明性とユーザーエクスペリエンスを約束します。

### **1. ゼロコスト、永久的無料の Serverless アーキテクチャ**

* **技術基盤：** プラットフォームは、**GitHub Pages** を直接利用してすべてのユーザーにサービスを提供し、**独自のサーバーを構築・運用しません**。
* **コスト優位性：** 運営コストはゼロに近づきます。プラットフォームはユーザーに対し**永久的無料利用**を約束でき、GitHubアカウントを登録するだけで参加できます。
* **分散化優位性：** 分散型アーキテクチャを採用しており、万が一GitHubがサービスを停止しても、すべてのコンテンツはユーザーのローカルリポジトリに保存されており、いつでもエクスポート可能で、コミュニティ運営を迅速に復旧できるため、IP資産の永続性が保証されます。

### **2. 透明性と最高のユーザーエクスペリエンスへの約束**

* **コードの透明性：** プラットフォームの全コードは**完全にオープンソース**とし、プラットフォーム運営の**絶対的な透明性**と公正性を保証し、隠れたロジックやバックドアが存在しないことを確実にします。
* **最高の閲覧体験：** プラットフォームは**永遠に広告を表示しない**ことを約束し、クリエイターと読者に、純粋で邪魔のない創作と閲覧環境を提供し、最良のユーザーエクスペリエンスを保証します。

### **3. 法務とIP資産管理**

* **CLA管理：** 財団は[CLA](CLA.md)（コントリビューターライセンス同意書）の署名を管理し、同意書をプライベートリポジトリに保存して、作者のプライバシーを保護します。
* **IPの統一管理：** 財団はCLAを通じて**すべてのIPの統一的な管理権と使用権を集中して取得**し、後の商業化運営に法的基盤を提供します。
* **CC BY-NC-SA 4.0：** すべての作品リポジトリはこのライセンスを使用しており、財団のみが商業開発を行えることが保証されます。
- クリエイティブ・コモンズ (CC): 知的財産は共有されます。
- 表示 (BY): すべての利用者は表示（クレジット）を行わなければなりません。
- 非営利 (NC): 誰でもこのIPを基に二次創作（ファンフィクション執筆、ファンアート作成）を行うことができますが、営利目的では使用できません。
- 継承 (SA): 他の人がこの作品を基に新たな作品を創作した場合、彼らは同じ[CC BY-NC-SA 4.0](LICENSE.md)ライセンスで共有しなければなりません。

### **4. 透明な非営利財務モデル**

* **資金源：** 主にプロジェクトの商業化運営で生じた収益の一部留保金や法的権利保護による賠償金、およびコミュニティからの寄付によって運営を維持します。
* **報奨制度：** 留保された剰余資金は**配当には使用せず**、**報奨基金**として、各プロジェクト理事会のノミネートとネットワーク全体の投票に基づき、ランキングに応じて選ばれた優秀なクリエイターに分配されます。
* **寄付管理：** 指定寄付は全額特定のプロジェクト理事会に引き渡します。非指定寄付は統一して会計処理し、剰余部分は報奨制度に基づき分配します。

## **三、クリエイターの中核的価値提案**

DIPCPプラットフォームは、クリエイターに従来のプラットフォームでは比類ない、6つの中核的優位性を提供し、クリエイターが **「単独での創作」** から **「集団による共創」** へと移行することを推進します。

| 優位性 | 中核的価値 | 説明 |
| :--- | :--- | :--- |
| **1. 集団共創の力** | **協働による強化** | 単独での限界を脱却し、チームの力を通じて共同創作を行います。協働によって生み出される作品は、より幅広い読者の嗜好を満たすことができ、後世に残る可能性が高まります。 |
| **2. コストと参入障壁の優位性** | **ゼロコスト協働** | 従来のプラットフォームが抱えるIP帰属権の調整困難さや、高額なプラットフォーム構築コストの問題を解決します。ユーザーは投資を一切必要とせず、ゼロコストでグローバルな協働に参加できます。 |
| **3. 規制のない創作の自由** | **想像力の解放** | プラットフォームは創作内容に対して公式な制限を行わず、クリエイターは想像力を存分に発揮し、多元宇宙を創造し、コンテンツの活力を刺激することができます。 |
| **4. 効率的な商業的実現** | **市場駆動** | **引用回数、正味いいね数**などのデータを通じて、優れた作品が埋もれることはありません。市場が明確に価値を認識し、商業的実現を直接的かつ効率的にします。 |
| **5. 曖昧化された作者の境界** | **最大規模のコミュニティ** | 作者と読者の境界を曖昧にし、世界中で唯一無二の、最大規模の自由文化創作コミュニティを形成します。 |
| **6. ミニマルな技術的ハードル** | **グローバルな使いやすさ** | 複雑なブロックチェーン技術やトークン発行（いわゆる「草刈り」）を使用せず、グローバル協働の参入障壁と法的リスクを大幅に低減します。 |

## **四、ガバナンスと商業化プロセス**

本システムのガバナンス構造は、**理事会が戦略を主導し、財団がサポートを提供する**形です。

### **1. IP価値実現メカニズム**

* **運営プロセス：** 各プロジェクト（例：あるIPユニバース）の**理事会**が財団に対して**文書による申請**を行った場合にのみ、財団は彼らを支援し、商業化運営（例：映画、ゲームの著作権販売）や法的権利保護を完了させます。
* **収益分配：** 運営が成功した後、財団は運営コストとして収益または賠償金のごく一部を留保し、**大部分の**収益は、プロジェクト理事会の合意に基づく分配方法（即ち、**[『多元宇宙フォークインセンティブモデル白書』](White_Paper_V2.1.md)**内のデータ式）に従って、クリエイターに分配されます。

### **2. ガバナンスの役割分担**

| 機関 | 中核的機能 |
| :--- | :--- |
| **プロジェクト理事会（Council）** | **プロジェクトの最高意思決定層。** 商業化の方向性を決定し、収益分配の重み付け（クリエイターの投票により決定）を設定し、コミュニティの基本的規範を維持します。 |
| **DIPCNPO** | **法務と行政サポート層。** CLA署名、IP統一管理、商業化委託、法的権利保護支援、資金の代理支払い、および報奨基金管理を担当します。 |
