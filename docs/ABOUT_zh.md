# **项目介绍：DIPCP——去中心化知识产权协作平台**
# **（DIPCP: Decentralized Intellectual Property Collaboration Platform）**

## **一、项目定位与愿景**

### **1. 组织主体：DIPCNPO (Decentralized intellectual property collaboration non-profit organization)**

* **性质：** 在新加坡成立的非盈利性法人组织。
* **使命：** 集中管理分散的知识产权（IP），为全球创作者提供一个稳定、法律健全且永久免费的协作环境。
* **愿景：** 赋能全球创作者，通过协作力量共同创作出能够流传于世的伟大知识产权作品。

### **2. 核心平台：DIPCP**

DIPCP 是一个基于 **GitHub Pages** 的去中心化协作平台，它彻底解决了传统协作模式中**高成本、IP 归属权模糊和中心化审查**的痛点。平台采用**“数据即裁判”**的激励机制，鼓励创作者通过分叉、引用和市场认可来孵化 IP 价值。

---

## **二、颠覆性的技术与运营架构**

DIPCF 基金会利用极简主义和分布式技术，打造了一个前所未有的零成本运营模型，并承诺极致的透明度与用户体验。

### **1. 零成本、永久免费的 Serverless 架构**

* **技术基石：** 平台直接利用 **GitHub Pages** 为所有用户提供服务，**不自建、不运维服务器**。
* **成本优势：** 运营成本趋近于零。平台可向用户承诺**永久免费使用**，只需注册一个 GitHub 账户即可加入。
* **去中心化优势：** 采用去中心化的分布式架构，即便 GitHub 停止运营，所有内容也保存在用户本地仓库中，可随时导出，快速恢复社区运作，保障了 IP 资产的持久性。

### **2. 透明度与最佳用户体验承诺**

* **代码透明度：** 平台的所有代码将**完全开源**，以保证平台运营的**绝对透明度**和公正性，确保无任何隐藏逻辑或后门。
* **最佳阅读体验：** 平台承诺**永远不打广告**，为创作者和读者提供一个纯净、无干扰的创作和阅读环境，保障最佳用户体验。

### **3. 法律与 IP 资产管理**

* **CLA 管理：** 基金会负责 [CLA](CLA.md)（贡献者许可协议）的签署，并将协议保存在私有化仓库中，保护作者隐私。
* **IP 统一管理：** 基金会通过 CLA **集中获得所有 IP 的统一管理和使用权**，为后续的商业化运作提供法律基础。
* **CC BY-NC-SA 4.0** 所有作品仓库都使用这个协议，确保了只有基金会才能进行商业开发。
- 知识共享 (CC): 知识产权是共享的。
- 署名 (BY): 任何使用者必须署名。
- 非商业性使用 (NC): 任何人都可以基于此IP进行二次创作（写同人小说、画同人图），但不能用于商业目的。
- 相同方式共享 (SA): 如果其他人基于此作品创作了新作品，他们必须以相同的[CC BY-NC-SA 4.0](LICENSE.md)协议分享。

### **4. 透明的非盈利财务模型**

* **资金来源：** 主要通过项目商业化运作中留存的小部分收益或法律维权赔偿金，以及接受社区捐助维持运营。
* **奖励机制：** 留存的盈余资金**不会用于分红**，将作为**奖励基金**，通过各项目理事会提名、全网投票，根据排名发放给入选的优秀创作者。
* **捐助管理：** 指定捐助全额交给特定项目理事会；非指定捐助统一做账，盈余部分按奖励机制发放。

## **三、创作者的核心价值主张**

DIPCP 平台为创作者提供了传统平台无法比拟的、六大核心优势，推动创作者从 **“单打独斗”**转向 **“集体共创”**。

| 优势 | 核心价值 | 描述 |
| :--- | :--- | :--- |
| **1. 集体共创的力量** | **协作赋能** | 摆脱单打独斗的局限，通过团队的力量共同创作。协作产出的作品能够满足更广泛的读者口味，更有潜力流传于世。 |
| **2. 成本与门槛优势** | **零成本协作** | 解决了传统平台难以协调 IP 归属和高昂的平台搭建成本问题。用户无需任何投资，零成本即可加入全球协作。 |
| **3. 无监管的创作自由** | **想象力释放** | 平台不对创作内容进行官方限制，创作者可以充分发挥想象力，创造多元宇宙，激发内容活力。 |
| **4. 高效的商业变现** | **市场驱动** | 通过**引用次数、净赞值**等数据，优秀作品不会被埋没。市场会清晰地识别价值，使商业变现变得直接且高效。 |
| **5. 模糊的作者边界** | **最大规模社区** | 将作者和读者的界限模糊化，形成全球独一无二、最大规模的自由文化创作社区。 |
| **6. 极简的技术门槛** | **全球易用性** | 不使用复杂的区块链技术或发行代币（不割韭菜），大大降低了全球协作的门槛和法律风险。 |

## **四、治理与商业化流程**

本系统的治理结构是**理事会主导战略，基金会提供支持**。

### **1. IP 价值变现机制**

* **运作流程：** 仅当各项目（例如，某个 IP 宇宙）的**理事会**向基金会提出**明文申请**时，基金会才会帮助他们完成商业化运作（如电影、游戏版权出售）或法律维权。
* **收益分配：** 运作成功后，基金会留取一小部分收益或赔偿金作为运营成本；**绝大部分**收益将根据项目理事会协议的分配方式（即**[《多元宇宙分叉激励模型白皮书》](White_Paper_V2.1.md)**中的数据公式），分发到创作者手中。

### **2. 治理分工**

| 机构 | 核心职能 |
| :--- | :--- |
| **项目理事会（Council）** | **项目最高决策层。** 负责决定商业化路径、设定收益分配权重（由创作者投票决定）和维护社区基本规范。 |
| **DIPCNPO** | **法律与行政支持层。** 负责 CLA 签署、IP 统一管理、商业化委托、法律维权协助、资金代发与奖励基金管理。 |
